process_book.py <входная_книга.txt> <выходная_книга.txt>
  разбивает книгу на отдельные слова
  выбирает только уникальные и сортирует по убыванию частоты встречания
  для каждого слова, если оно не присутствует в словарях знакомых и незнакомых слов, с помощью Яндекс получает перевод и добавляет слово в словарь незнакомых слов
  формирует текст книги, используя слова из словарей
